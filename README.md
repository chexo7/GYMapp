# Roxborough Training Log (Next.js)

Migrated from the original static HTML into a Next.js 14 app, optimized for deployment on Vercel.

## Scripts
- `npm run dev` - iniciar el servidor de desarrollo.
- `npm run build` - compilar la app para producción.
- `npm start` - ejecutar la build generada.
- `npm run lint` - ejecutar ESLint.

## Notas
- Las imágenes remotas se permiten desde `lh3.googleusercontent.com`.
- La interfaz conserva el flujo de login, dashboard e historial con datos mock.
